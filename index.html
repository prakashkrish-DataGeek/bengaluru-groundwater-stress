GWPZ Formula

Analytical Hierarchy Process (AHP) weighted index, peer-reviewed weights from hydrogeological literature.

// Groundwater Potential Zone composite

GWPZ =  (w1 × LULC_score)  + (w2 × Slope_score)  +
        (w3 × NDVI_score)  + (w4 × Lineament_score)

// Weights (sum = 1.00):
w1 = 0.30  — LULC recharge suitability
w2 = 0.25  — Terrain slope (flat = high recharge)
w3 = 0.25  — NDVI (dense veg = good infiltration)
w4 = 0.20  — Lineament/fracture density

// Each layer reclassified: 1 (Very Low) → 5 (Very High)
// GEE implementation (Script 03)
var gwpz = lulcScore.multiply(0.30)
  .add(slopeScore.multiply(0.25))
  .add(ndviScore.multiply(0.25))
  .add(lineamentScore.multiply(0.20))
  .rename('GWPZ');

Key Findings

Preliminary observations from the multi-sensor groundwater stress analysis.

🏙️

Urban Core = Lowest GWPZ

Central Bengaluru (MG Road, Shivajinagar, Koramangala) shows GWPZ Class 1–2 due to high imperviousness and steep runoff.

🌿

Peripheral Greens = High Recharge

Bannerghatta, Kengeri, Yelahanka periphery and Turahalli forest retain GWPZ Class 4–5 with intact vegetation cover.

🌡️

UHI Amplifies Stress

Whitefield and Electronic City summer LST exceeds 44°C — 8–12°C above the rural baseline, compounding groundwater demand.

📉

Backscatter Instability Hotspots

Elevated SAR temporal variance detected in Bommanahalli, KR Puram and Sarjapur corridors — consistent with documented subsidence.

💧

Drains Reduce Recharge

Within 250m of primary SWD corridors, effective recharge potential drops ~30% as surface runoff is channeled away.

🪨

Fractures Control Deep Storage

NNW-SSE lineaments (Closepet Granite trend) correlate with deeper aquifer yields — key targets for artificial recharge wells.
